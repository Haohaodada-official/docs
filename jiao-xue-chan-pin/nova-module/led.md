# LED

## 概述

Nova的LED模块主要实现LED灯的亮度的控制。

## 参数

* 尺寸：40x23mm
* 工作电压：+3.3-5V
* 类型：数字模式
* 接口模式：2510-3p
* 引脚定义：1-控制端 2-电源 3-地

## 接口说明

* 可用端口： A0、A1、A2、A3、S0、S1、S2、S3
