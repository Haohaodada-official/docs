# 单按键模块说明

## 概述
Nova的单按键模块，可输出一个高低电平，按键没有按下时是输出高电平，按下时为输出低电平，从而控制其他的外设等。

## 参数
- 尺寸：40x23mm
- 电源要求：+5V
- 类型：数字模式
- 接口模式：2510-3p
- 引脚定义：1-控制端 2-电源 3-地

## 接口说明
- 可用端口： A0、A1、A2、A3、S0、S1、S2、S3
